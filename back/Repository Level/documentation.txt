Функции и подаваемые аргументы:

create(adress: str, password: str)
Функция create создает новую запись содержащую почту пользователя (adress), являющуюся уникальным значение в таблице, и пароль (password)

dlete(adress: str)
Функция delete удаляет запись пользователя с почтой adress

find(adress:str)
Функция find возвращает объект содержащий данные о пользователе, переменная adress содержит почтовый ящик, password содержит пароль
